\section{Code Structure}

The project is organized in modular fashion:

\begin{lstlisting}[caption=Project Structure]
data_mining_final_project/
|-- main.py                    # Main execution script
|-- data_preprocessing.py      # Data loading and preprocessing
|-- clustering_algorithms.py   # Base clustering implementations
|-- ensemble_methods.py        # Ensemble clustering methods
|-- evaluation.py             # Metrics and stability analysis
|-- interpretation.py         # Cluster interpretation functions
|-- plots.py                  # Visualization functions
|-- utils.py                  # Utility functions and metrics
|-- README.md                 # Project documentation
|-- requirements.txt          # Python dependencies
|-- Mall_Customers.csv        # Dataset
|-- results/                  # Output results
\-- plots/                    # Generated visualizations
\end{lstlisting}
