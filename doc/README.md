# Final Submission - Customer Segmentation Clustering

Questa cartella contiene tutti i materiali richiesti per la consegna finale del progetto di data mining.

## Contenuto
- `main.pdf` — Paper in formato IEEE
- `presentation_slides_text_english.md` — Testo e struttura della presentazione
- `metodologie_sviluppo.md` — Spiegazione metodologie, algoritmi, domande frequenti
- `requirements.txt` — Dipendenze Python
- `main.py` — Script principale per eseguire l'analisi
- `data_preprocessing.py`, `clustering_algorithms.py`, `ensemble_methods.py`, `evaluation.py`, `interpretation.py`, `plots.py`, `utils.py` — Moduli core
- `Mall_Customers.csv`, `Customer_Personality.csv`, `Wholesale_Customers.csv` — Dati usati
- `plots/` — Grafici principali

## Istruzioni
1. Leggere il paper (`main.pdf`) per la descrizione completa.
2. Eseguire `main.py` per riprodurre l'analisi (richiede le dipendenze in `requirements.txt`).
3. Consultare `metodologie_sviluppo.md` per dettagli su scelte metodologiche e risposte a domande frequenti.
